# cxr-labeller frontend

React + TypeScript + Vite, wired up to also run as a desktop app via Tauri.

## Web dev

```sh
npm install
npm run dev
```

## Desktop dev (requires Rust: https://www.rust-lang.org/tools/install)

```sh
npm install
npm run tauri dev
```

## Build

```sh
npm run build        # web build -> dist/
npm run tauri build  # desktop app bundle
```
